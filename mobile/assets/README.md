# Assets Directory

Bu klasöre uygulama icon'larını ve splash screen'leri ekleyin:

- `icon.png` (1024x1024) - Ana uygulama icon'u
- `splash.png` (1284x2778) - Splash screen
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (48x48) - Web favicon

## Hızlı Başlangıç

Geliştirme için icon'lar opsiyoneldir. Production build için gerekli olacaktır.

Icon oluşturmak için:
1. Online tool kullanın: https://www.appicon.co/
2. Veya Expo'nun icon generator'ını kullanın: `npx expo install expo-asset`
